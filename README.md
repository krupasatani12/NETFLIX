# 🎬 Netflix Clone - High Performance Landing Page

Welcome to the **Netflix Clone**! This project is a high-fidelity recreation of the Netflix landing page, featuring premium UI/UX, advanced CSS animations, and modern JavaScript interactivity. It's designed to be lightweight, responsive, and visually stunning.

---

## 🚀 Live Preview & Features

### ✨ Key Features
- **Dynamic Header**: A sticky navbar that transitions its background and size smoothly on scroll.
- **Scroll Reveal Animations**: Sections elegantly fade and slide into view as the user navigates down the page.
- **Interactive FAQ Accordion**: A custom-coded accordion logic that allows only one question to be active at a time for a clean UX.
- **Premium Sign In Modal**: A stunning glassmorphism-based login popup with form validation.
- **Hero Form Validation**: Real-time feedback for email subscriptions in the hero section.
- **Device-Specific Optimization**: Pixel-perfect responsiveness across Desktop, Tablet, and Mobile devices.
- **Optimized Video Backgrounds**: Efficient use of HTML5 video tags for an immersive experience.

## 🛠️ Project Architecture

```bash
NETFLIX/
├── assets/
│   ├── img/          # SVGs, Favicons, and Backgrounds
│   └── videos/       # Dynamic video assets for content sections
├── index.html        # Main semantic layout & Modal structure
├── style.css         # Modern CSS with variables & keyframe animations
└── script.js         # Core logic (Accordion, Modal, Observer)
```

---

## 💻 Tech Stack

- **HTML5**: Semantic elements and video integration.
- **CSS3 (Advanced)**: 
  - Custom variables (`:root`)
  - Flexbox and Grid layouts
  - Keyframe animations & transitions
  - Backdrop filters (Glassmorphism)
- **JavaScript (ES6+)**:
  - `IntersectionObserver` for reveal animations
  - DOM manipulation for Modal & Accordion
  - Form validation & Event listeners
